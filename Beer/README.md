🍺 Beer Selector Pro: Sistema de Recomendación de Cervezas

🎯 Objetivo del Proyecto
Desarrollar un sistema analítico y una aplicación interactiva capaz de clasificar estilos de cerveza basándose en sus propiedades fisicoquímicas y perfil sensorial. Esta herramienta permite a los usuarios identificar su cerveza ideal, optimizando la toma de decisiones mediante filtros de negocio y un modelo de aprendizaje no supervisado.

🛠️ Stack Tecnológico
Lenguaje: Python

Ingeniería de Datos: Requests, BeautifulSoup (Web Scraping)

Procesamiento y Análisis: Pandas, NumPy

Visualización: Matplotlib, Seaborn, Plotly, Streamlit

Machine Learning: Scikit-Learn (K-Means Clustering)

📊 Hallazgos Clave
Segmentación Inteligente: Mediante el algoritmo K-Means, logramos agrupar 81 estilos de cerveza en clústeres basados en su amargor (IBU), contenido de alcohol (ABV) y perfil de fermentación.

Optimización de Búsqueda: La aplicación permite filtrar resultados con una tolerancia de +/- 1.5% ABV, lo que garantiza recomendaciones personalizadas y precisas para el perfil del usuario.

Interfaz orientada al usuario: Diseño mobile-first implementado con Streamlit para una experiencia fluida e intuitiva.
